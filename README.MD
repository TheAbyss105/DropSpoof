# DropSpoof

Fake Airdrop for android phones!

Requirements:

- NFC Compatibility
- Spare NFC Tag

## Installation

The APK file is in releases, but you can build it yourself if you want.

## Usage

Click a file/URL/contact card, click share and then DropSpoof.

Turn NFC on, tap the tag to your phone if it's not in the phone case already, and then turn it back off once you feel a buzz.

Tap the area where the tag is to another phone! For iphones, it's the top bar.

NOTE: Sharing files requires users to be on same network, and contact cards are stripped of their profile pictures.
